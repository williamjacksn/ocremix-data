OC ReMix Data

This project provides data about songs published by OverClocked ReMix. The data is available in JSON and SQLite format.

ReMix data

Example URL for data about a single ReMix:

  • https://williamjacksn.github.io/ocremix-data/remix/OCR00001.json

Tag data

Example URL for data about a single tag:

  • https://williamjacksn.github.io/ocremix-data/tag/electronic.json

SQLite

Download all data as a SQLite database from this url:

  • https://williamjacksn.github.io/ocremix-data/ocremix-data.db
